## 1. GCD 알고리즘

### (1) 알고리즘 설명
유클리드 호제법을 사용하여 최대공약수를 계산하였다.

### (2) 코드
```c
int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}
```
